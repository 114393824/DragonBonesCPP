DragonBonesCPP

DragonBones C++ Library

# Supported engines

## [cocos2d-x][1]

Please get the cocos2d-x source codes and copy them to **engines** directory.

* For cocos2d-x 2.x: engines/cocos2d-x-2.x
* For cocos2d-x 3.x: engines/cocos2d-x-3.x
